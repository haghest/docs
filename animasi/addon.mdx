## Cara Install Addons
Cara menginstall Add-ons di Blender
1. Klik Edit, lalu pilih Preferences
<img  style={{ borderRadius: "0.5rem" }} src="/images/install addon/14-38-21_30-03-2025_blwCU4nzzu.png"/>
2. Pilih menu Add-ons, lalu klik Install
<img  style={{ borderRadius: "0.5rem" }} src="/images/install addon/23-38-21_30-03-2025_grKkWJjabf.png"/>
<Info>
  **Info**: Plugin Blender tidak perlu diextract, biarkan dalam format ```.zip```.
</Info>
