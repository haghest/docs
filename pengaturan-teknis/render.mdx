## Render menggunakan EEVEE
Semua animasi dirender menggunakan EEVEE agar lebih cepat. Pastikan bukan Cycles.
## 1080p
Pastikan sudah menggunakan resolusi **1080p** pada menu **Output Properties**
<img style={{ borderRadius: "0.5rem" }} src="/images/resolusi_blender.png" />
## Color Management 
Ubah View Transform dari Filmic ke **Standard** pada halaman Render Properties scroll kebawah ubah seperti setting ini. Ini agar tidak perlu melakukan color grade lagi dan mengeluarkan warna kontras kartun pada render
<img style={{ borderRadius: "0.5rem" }} src="/images/color_blender.png" />

## Menggunakan .mp4
<img style={{ borderRadius: "0.5rem" }} src="/images/render_blender.png" />
Pada menu Output Properties, pastikan pilih dimana file akan disimpan terlebih dahulu. Lalu ubah File format ke **FFmpeg Video** dan buka tab **Encoding** lalu pilih Container **MPEG-4**
